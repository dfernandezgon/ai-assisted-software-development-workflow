:# AI-Assisted Software Development Workflow

A human-centered framework for thinking, building, and maintaining software in the age of AI.

## About

This repository contains a practical engineering workflow for using AI tools in real software projects while preserving human responsibility, version control, documentation, testing, security and review.

The workflow is based on real migration, maintenance, refactoring and development work involving conversational AI, repository-level AI agents, Git, GitHub and structured project memory.
